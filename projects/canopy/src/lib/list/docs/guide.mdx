## Standard list

### Unordered list

For a bulleted unordered list, use standard HTML:

```html
<ul>
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ul>
```

### Ordered list

For an ordered list, use standard HTML:

```html
<ol>
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ol>
```

#### Ordered list with expressive styling

The `lgListWithExpressiveStyling` directive can be used to replace the default counter of an ordered list with one styled to be expressive:

```html
<ol lgListWithExpressiveStyling>
  <li>List item 1</li>
  <li>List item 2</li>
  <li>List item 3</li>
</ol>
```

## List with icons

### Usage

Import the list with icons components in your application:

```ts
import {
  LgListWithIconsComponent,
  LgListWithIconsItemComponent,
} from '@legal-and-general/canopy';
```

Register the icons as explained in the [iconography guide](./?path=/docs/foundations-ui-icon-guide--docs).

### Item theme

Set the item marker theme per item via `variant`: `mono`, `positive` and `negative`

These semantic values map to the corresponding marker colour tokens.

See each theme in the examples:

- `mono`: [Unordered list](./?path=/story/components-list-with-icons-examples--unordered-list)
- `mono`: [Ordered list](./?path=/story/components-list-with-icons-examples--ordered-list-with-numbers)
- `positive` and `negative`: [Considerations list](./?path=/story/components-list-with-icons-examples--considerations-list)

### Sizes

Lists support two sizes:

- **default**: standard typography scale and spacing
- **large**: larger typography and increased spacing for emphasis

### Indentation

List items can be nested across two levels: parent and child.

Child items are added by placing a nested list inside a parent list item:

```html
<ul lg-list-with-icons>
  <li lg-list-with-icons-item iconName="help">List item 1</li>
  <li lg-list-with-icons-item iconName="idea">
    List item 2
    <ul lg-list-with-icons>
      <li lg-list-with-icons-item iconName="checkmark">Child item 1</li>
      <li lg-list-with-icons-item iconName="checkmark">Child item 2</li>
    </ul>
  </li>
  <li lg-list-with-icons-item iconName="house-outline">List item 3</li>
</ul>
```

### Icon customisation

Icons are customisable via the `iconName` input on each `lg-list-with-icons-item`. Recommended icons for common use cases:

- `checkmark`
- `checkmark-spot-filled`
- `bullet-point`
- `bullet-square`
- `bullet-feature`
- `bullet-dash`
- `bullet-circle`
- `progress-to-do`
- `cross`
- `crossmark-spot-filled`

Marker colour can be customised per item using the `variant` input.

For ordered lists, marker numbers can be set per item using the `number` input.

### LgListWithIconsComponent

The container for list items with optional icon support.

Item marker theme is controlled on each `LgListWithIconsItemComponent` via its
`variant` input.

#### Inputs

| Name   | Description                         | Type     | Default   | Required |
| ------ | ----------------------------------- | -------- | --------- | -------- |
| `size` | The list size: `default` or `large` | `string` | `default` | No       |

### LgListWithIconsItemComponent

Individual list item with icon support.

#### Inputs

| Name       | Description                                         | Type     | Default              | Required |
| ---------- | --------------------------------------------------- | -------- | -------------------- | -------- |
| `iconName` | The name of the icon to display                     | `string` | `bullet-point`       | No       |
| `number`   | Ordered marker number for the item                  | `number` | Auto (list position) | No       |
| `variant`  | The marker theme: `mono`, `positive`, or `negative` | `string` | `mono`               | No       |
